Project Overview
The AI Assignment Checker is an AI-powered automation workflow built using Make.com that evaluates student assignments based on content quality, relevance, structure, and clarity.
It automates the assignment review process by generating performance scores, identifying strengths and improvement areas, and delivering a structured feedback report via email.

------------------------------------------------------------------------------------------------------------------------------------
Key Features
AI-based assignment evaluation
Content quality and relevance analysis
Automated scoring system
Identification of strengths and improvement areas
Structured feedback report generation
Automatic email delivery of evaluation results

------------------------------------------------------------------------------------------------------------------------------------

How It Works

1. User submits an assignment in google docs link format.
2. Assignment content is extracted
3. AI analyzes content quality, structure, and relevance
4. Performance score and evaluation summary are generated
5. Strengths and improvement areas are identified
6. A structured feedback report is created
7. The report is automatically sent via email

-------------------------------------------------------------------------------------------------------------------------------------

Tech Stack
Make.com for workflow automation
AI model - Gemini or OpenAI for content evaluation
Gmail module for automated email delivery
Text parsing and content extraction tools
Prompt engineering for consistent evaluation output
